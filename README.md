<!-- DESCRIPTION/ -->

React starter kit that is more minimal than essential

<!-- /DESCRIPTION -->


## Usage

``` bash
# Setup
git clone https://github.com/balupton/minimal-react.git
cd minimal-react
npm install

# Once-off compiles
npm run-script compile

# Development build
npm start

# Testing
npm test

# Updating Meta Files
npm run-script meta
```
